# Text Summarizer

A simple Node.js TypeScript application using AI SDK 5 to summarize text via command line.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Set your Vercel API key as an environment variable:
```bash
export AI_GATEWAY_API_KEY=your_api_key_here
```

## Usage

```bash
npm start "Your text to summarize goes here"
```

### Example

```bash
npm start "Artificial intelligence is transforming the way we work and live. Machine learning algorithms can now process vast amounts of data and identify patterns that humans might miss. This technology is being applied in healthcare, finance, transportation, and many other industries."
```

## Requirements

- Node.js 18+
- OpenAI API key
